# CryptoAlgorithms
**New version of the repository 'Crypto'. The code has been refactored.**

Now each cipher is represented in class.
```
|-- CryptoAlgorithms
  |-- tools
      |--data_convert.py（数据类型转换）
      |-- math_calculate.py（数学计算）
  |-- classical
    |-- Affine.py（仿射密码）
    |-- MonoSub.py（单表替代密码）
    |-- Vernam.py（弗纳姆密码）
    |-- Vigenere.py（维吉尼亚密码）
```
